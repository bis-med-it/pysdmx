GDS Connector
=============

Overview
--------
The GDS connector allows you to perform REST queries over the SDMX Global Discovery Service.

The SDMX Global Discovery Service (SGDS) is an official SDMX tool designed to help users discover and access SDMX data
and metadata sources across multiple registries.
It enables efficient discovery and sharing of SDMX artefacts,
offering both a user interface and API for endpoint and resource resolution.
