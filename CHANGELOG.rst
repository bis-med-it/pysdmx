Changelog
=========

All notable changes to this project are listed in the 
`releases section <https://github.com/bis-med-it/pysdmx/releases>`_ .

The format is based on `Keep a Changelog <https://keepachangelog.com/en/1.0.0/>`_,
and this project adheres to 
`Semantic Versioning <https://semver.org/spec/v2.0.0.html>`_.

However, please note that this project may at times offer experimental classes 
or functions. Experimental classes and functions will be clearly indicated as 
such in the documentation. They are provided as a preview and are not considered
part of the stable API. These classes may change, break, or be removed at any time
without prior notice. Changes to experimental classes or functions will not trigger
a major version increment under semantic versioning. Use them at your own risk,
especially in production environments.
